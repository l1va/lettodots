# lettodots
Convert text to points for drawing by manipulator

### Run:
```
python main.py Hello world!
``` 
or multiline example: 
```
python main.py
``` 

Result is in output.csv

### Simulator (Turtle):
```
python turtle_sim.py Hello world!
```
or multiline example: 
```
python turtle_sim.py
```
